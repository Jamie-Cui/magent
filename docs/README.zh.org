欢迎阅读 Magent 文档。Magent 是一个 Emacs Lisp AI coding agent，具备 multi-agent 架构、permission-based tool access，并通过 [[https://github.com/karthink/gptel][gptel]] 集成 LLM。

* 文档维护标准

新增或修改文档时：

- 用户稳定入口优先更新根目录 ~README.org~ 。
- 开发者文档放在 ~docs/~ 。
- ~docs/~ 下使用 Org 作为源码；生成的 HTML 是 ~_site/~ 下的构建产物。
- 新增英文文档时同步新增中文镜像页，并在导航和本索引中登记。
- 中断开发前更新稳定文档或活动任务笔记，让其他机器能从 git 恢复上下文。
